## 📖 About The Project

**Snack Station** is a Java-based vending machine software developed as a **First Year – First Semester** project. The system simulates the functionality of a real-world vending machine, allowing users to browse available snacks, select products & make purchases.

---

## ✨ Features

### 👤 Customer Features

* View available snacks and beverages
* Select products from the vending machine
* Purchase items
* Display product prices
